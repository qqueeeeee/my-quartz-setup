# Fishao Server

This is the idea of building a Fishao-compatible backend by reverse engineering the original protocol and recreating enough of the server side to make it actually work.

This is one of those projects I really like because it is way more interesting than normal CRUD stuff. It forces me to care about protocols, packet structure, backend architecture, and how games actually talk to services.

## Why it matters to me

- it fits the hacker / tinkerer side of my interests way more naturally
- it connects low-level curiosity with actual software engineering work
- it is the kind of thing that reminds me why I like programming in the first place
