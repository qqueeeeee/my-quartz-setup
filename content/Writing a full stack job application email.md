# Writing a full stack job application email

This is a small practical note on writing direct job application emails without overcomplicating them.

## Notes

I used this chat for two different things. The first part was writing a proper email for a Full Stack Developer application at Ayrin Digital. The first draft leaned too much on project details, so I pushed it back toward my background and skills instead because the resume would already be attached. Claude rewrote it into something shorter and more personal. The second part was making a strong prompt for another AI to add a full authentication system to an existing full-stack project. That part was for a friend, not for me directly. I asked for the prompt to avoid hardcoding the stack and instead tell the AI to detect the stack from the existing project files. The final prompt covered backend stuff like user/token models, auth endpoints, JWT, and security measures, and frontend stuff like login/register/profile pages, auth context, protected routes, and role-based access. At the end I also had Claude help fill out a job application form from my resume. This was a very practical chat overall, so I kept correcting anything that felt too generic or too far from what I actually wanted.
