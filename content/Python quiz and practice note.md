# Python quiz and practice note

This is a small note around using quizzes and lightweight exercises to sharpen up my [[Python]] thinking.

## Notes

One of the easiest ways to figure out whether I actually know something in [[Python]] is to get quizzed on it instead of just rereading notes.

That matters because passive review feels productive way more often than it actually is.

Quizzes and short code prompts are useful for:

- checking whether I really understand syntax instead of just recognizing it
- spotting weak points quickly
- keeping practice light enough that I will actually do it

So the value here is not some giant study system. It is just having a low-friction way to pressure-test what I know.
