# Create things you wish existed

One thing I keep coming back to is this:

if I want to build good stuff, I should probably build the kind of thing I wish already existed.

That sounds obvious, but I think it matters a lot because it solves a bunch of problems at once.

- it makes the work feel more real
- it gives me actual taste instead of random feature ideas
- it makes it easier to care long enough to keep going

That is part of why projects like [[SelfForge]] or [[Quartz]] matter to me more than some generic portfolio project ever would. They are both trying to become things I would genuinely want to use.

I think this is also tied pretty closely to [[Find your own theme..]].

If I am building something I wish existed, it is way easier for it to have a real point of view instead of feeling like copy-pasted product design.
