The way I use [[Obsidian]] is based off of [Steph Anglo's Workflow](https://stephango.com/vault), it uses Properties, Tags instead of Folders, and you find files through the Quick Switcher or the command pallet. I use [[Neovim|vim]] motions, as I prefer using only my keyboard — more about that in my [[Keyboard Centric Workflow]]. 

I use a bunch of [[Templates]] to create notes, as they make it easy by inserting properties automatically, which saves a lot of time and makes me actually organize my notes as they can be organized fairly easily.
